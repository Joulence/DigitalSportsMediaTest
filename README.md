## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Joulence/DigitalSportsMediaTest.git
   ```

2. Navigate to the project directory:

   ```bash
   cd DigitalSportsMediaTest
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

  ```bash
  npm run dev
  ```
